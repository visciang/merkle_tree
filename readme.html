    MerkleTree

    (Sparse) Merkle Tree.

    MerkleTree implements Enumerable and Collectable protocols.

    Quick start 

    mt1 = MerkleTree.new(a: 0, b: 2, c: 3)

mt2 =
    mt1
    |> MerkleTree.put(:a, 1)
    |> MerkleTree.delete(:c)
    |> MerkleTree.put(:d, 4)

3 = MerkleTree.size(mt1) = MerkleTree.size(mt2)
false = MerkleTree.merkle_equal?(mt1, mt2)
[{:add, :d}, {:upd, :a}, {:del, :c}] = MerkleTree.merkle_diff(mt1, mt2)
%{c: 3, a: 0, b: 2} = Map.new(mt1)
